# Frustum 倒角功能说明

## 概述
为Frustum（截锥体）添加了倒角功能，可以在上下底面与侧面之间创建平滑的倒角过渡。

## 新增参数

### 倒角参数 (Chamfer Category)
- **ChamferRadius**: 倒角半径，控制倒角的大小
- **ChamferSegments**: 倒角细分段数，控制倒角的平滑度
- **bEnableChamfer**: 是否启用倒角功能

## 功能特点

1. **上下底面倒角**: 在顶部和底部表面与侧面之间创建倒角
2. **顶面收缩**: 顶面会收缩高度和半径
3. **底面固定**: 底面保持原来的高度，Z坐标不变，只有半径收缩
4. **环形弧面**: 生成连续的环形倒角弧面，确保几何连续性
5. **精确几何**: 通过环形顶点阵列确保几何的精确性
6. **可调节参数**: 可以调整倒角半径和细分段数
7. **平滑过渡**: 倒角提供平滑的几何过渡
8. **可选功能**: 可以通过bEnableChamfer开关控制是否启用倒角

## 使用方法

1. 在Frustum Actor的Details面板中找到"Chamfer"分类
2. 勾选"bEnableChamfer"启用倒角功能
3. 调整"ChamferRadius"设置倒角大小
4. 调整"ChamferSegments"设置倒角平滑度

## 技术实现

- 使用FChamferGeometryGenerator生成倒角几何
- 倒角生成器在几何管理器中优先执行
- 当启用倒角时，顶部和底部几何生成器会被跳过，由倒角生成器处理
- 倒角生成器生成收缩后的主面和环形倒角弧面
- 底面保持原来的高度，Z坐标不变，只有半径收缩
- 侧面几何生成器只调整顶部高度，底部高度保持不变
- 倒角弧面通过环形顶点阵列生成，确保平滑过渡和几何连续性

## 注意事项

- 倒角半径不应超过Frustum的半径
- 增加倒角段数会提高几何复杂度
- 倒角功能与弯曲、端面等功能兼容
